# Sistema de gestión de tareas diarias
Proyecto en fase de desarrollo que busca gamificar las tareas diarias tomando inspiración de mecánicas de videojuegos

## ✅ Características basicas futuras
- Agregar tareas personalizadas - Crea tus propias misiones diarias
- Experiencia por tarea - Cada tarea otorga XP específico
- Bonus de completación - XP extra por terminar todas las tareas del día
- Interfaz visual atractiva - Diseño inspirado en videojuegos
